## ✨ コンセプト

時間が経つと自動でポイントが溜まる。

🛁 おふろボタンを押すとリセット。

放置すると危険度が上がる。  
でも続ければ「神清潔モード」に突入。

さらに——  
清潔を7日連続で達成すると「ごほうび30日履歴」が解放されます。

ゆるく煽られ、  
ゆるく達成感を得る、  
ゆめかわ習慣ジョークアプリです。

## 🧼 清潔連続の仕様（重要）

清潔連続は「🛁を押した日」を清潔達成日として数えます。

- 同じ日に何回🛁を押しても +1 はされません（1日1回まで）
- 前回の清潔達成日が「昨日」なら連続 +1
- 1日でも空いたら連続は途切れ、次は 1 から再スタート

例：

- 1日 🛁 → 1日目
- 2日 🛁 → 2日目
- 3日 何もしない（空く）
- 4日 🛁 → 1日目にリセット

## 🌸 神清潔モードとは？

清潔連続 7日達成で発動。

- ヘッダー下に神清潔バッジ表示
- 達成時にトースト演出
- 危険度ゲージが“ピンク浄化カラー”に変化
- 表示ラベルが進化
- 🎁 30日履歴がごほうび解放

継続すると、UIがほんのり神モードになります。

---

## 🎁 ごほうび30日履歴

通常は「7日履歴」のみ表示。

神清潔（清潔連続7日以上）になると  
自動で「30日履歴」に拡張されます。

押す操作はありません。  
がんばると、勝手に増えます。
