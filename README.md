# 🫧 ちんかすカウンター (v0.4.0)

ゆめかわ清潔で時間と戦うジョークPWA。

👉 https://skurosawa.github.io/chinkasu-pwa/

---

## ✨ コンセプト

時間が経つと自動でポイントが溜まる。

🛁 おふろボタンを押すとリセット。

放置すると危険度が上がる、
ゆるく煽られる習慣系ジョークアプリ。

- オフライン対応
- ホーム画面に追加可能（PWA）
- 清潔連続日数を記録
- 危険度ゲージで可視化

---

## 🚀 主な機能（v0.4）

- ✅ 1時間で+1ポイント（最大48）
- ✅ 🛁ボタンで即リセット
- ✅ 危険度ゲージ表示（0〜100%）
- ✅ 清潔連続日数カウント
- ✅ 最長清潔記録
- ✅ 🛁リセット履歴保存（直近表示）
- ✅ オフライン起動対応
- ✅ iPhone動作確認済み

---

## 📱 使い方

1. Safariでアクセス  
2. 「共有」→「ホーム画面に追加」
3. アプリのように起動可能
4. 時間経過でポイントが増えます
5. 🛁ボタンでリセットできます

※ オフラインでも動作します

---

## 📊 危険度仕様

- 0〜3：きらきら清潔✨
- 4〜11：そろそろおふろ🫧
- 12〜23：リセット推奨🙂
- 24〜：危険ゾーン⚠️
