# 🫧 ちんかすカウンター (v0.3.0)

ゆめかわ清潔で毎日きろくするジョークPWA。

👉 https://skurosawa.github.io/chinkasu-pwa/

---

## ✨ コンセプト

毎日の「きろく」を通して自分をゆるく煽る、
ゆめかわ清潔チャレンジアプリ。

- オフライン対応
- ホーム画面に追加可能（PWA）
- 清潔連続日数を自動計算

---

## 🚀 主な機能

- ✅ 今日のカウント（+1ボタン）
- ✅ 累計カウント表示
- ✅ 清潔連続日数
- ✅ 最長清潔記録
- ✅ オフライン起動対応
- ✅ iPhone動作確認済み

---

## 📱 使い方

1. Safariでアクセス  
2. 「共有」→「ホーム画面に追加」
3. アプリのように起動可能
4. オフラインでも動作します
